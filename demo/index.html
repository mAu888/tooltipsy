Naked tooltip

Try out this very basic unstyled tooltip. $('#tooltip-naked').tooltipsy();

Basic Tooltip

Added some color and styling to this tooltip. $('#tooltip-basic').tooltipsy({ classes: { tipsy: 'tipsy-tooltip-basic', tip: 'tip-tooltip-basic', pointer: 'pointer-tooltip-basic' } });

Basic Tooltip to the left

Yeah, you may position it where ever you want. In this case I choose left. $('#tooltip-basic-left').tooltipsy({ classes: { tipsy: 'tipsy-tooltip-basic', tip: 'tip-tooltip-basic', pointer: 'pointer-tooltip-basic' }, offset: [-1, 0] });

Basic Tooltip to the right

So, let's try the right side. $('#tooltip-basic-right').tooltipsy({ classes: { tipsy: 'tipsy-tooltip-basic', tip: 'tip-tooltip-basic', pointer: 'pointer-tooltip-basic' }, offset: [1, 0] });

Foo bar

You may take any html element as tooltipsys base element. Check this out. For the sake of usuability you may change the delay out value to e.g. 1000. $('#tooltip-custom-el').tooltipsy({ offset: [0, 1], pointer: true, container: '#tipsy', classes: { tipsy: 'tipsy-tooltip-custom-el', tip: 'tip-tooltip-custom-el', pointer: 'pointer-tooltip-custom-el' }, delayOut: 1000 });
